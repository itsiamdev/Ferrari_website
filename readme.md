# Ferrari Website

This project is a static website dedicated to showcasing Ferrari cars. It features a collection of high-quality images of various Ferrari models, including the SF90, Portofino, Roma, and Formula 1 cars. The site is built using HTML, CSS, and JavaScript for a simple, responsive design.

Visit the site: [https://iam269.github.io/Ferrari_website/](https://iam269.github.io/Ferrari_website/)
